02 · AGENT

AI 自动化

几套每天自动在跑的 agent 工作流，结果推到手机上。

多 Agent 系统 OPENCLAW

5 个 agent 各管一摊：求职、内容、财经、健康、运维，加上十来个定时任务，从早晨的市场预测到周末的反思报告。基于开源项目 OpenClaw 搭的。

小红书调研

给一个主题，自动打开手机端的小红书 app 搜索、一条条读笔记，最后汇总成一篇调研报告。快档 5 到 10 分钟，深档 15 到 30 分钟。

每日预测 MIROFISH

开源预测引擎 MiroFish，我给它接了条管线：每天凌晨把当天新闻自动跑成趋势预测，早上发出来，再用记分板回头对答案。

市场分析 DIGITAL ORACLE

跟着 B 站 Komako 的开源思路：少看新闻头条，看市场价格反推趋势。每天自动跑换汇分析，也让一个 agent 在 ASX 模拟炒股比赛里自动看盘下单（模拟盘，战绩中游）。
